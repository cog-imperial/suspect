Directed Acyclic Graph¶

class suspect.dag.ProblemDag(name=None)¶

The optimization problem represented as Directed Acyclic Graph (DAG).

The vertices in the DAG are sorted by depth, defined as

\[\begin{split}d(v) = \begin{cases} 0 & \text{if } v \text{ is a variable or constant}\\ \max\{d(u) | u \in c(v)\} & \text{otherwise} \end{cases}\end{split}\]

where \(c(v)\) are the children of vertex \(v\).

Attributes:
name : str

the problem name.

variables : dict

the problem variables.

constraints : dict

the problem constraints.

objectives : dict

the problem objectives.

vertices : iter

Return an iterator over the problem vertices.
